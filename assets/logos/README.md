# Logos

Logo and mark masters. SVG is the canonical format; raster derivatives generated as needed.

## Per-brand expected files

### HensPham
- `hensph-wordmark-mono.svg` — primary wordmark, single-colour
- `hensph-wordmark-onlight.svg` — light-background variant
- `hensph-wordmark-ondark.svg` — dark-background variant

### ConnexusMinds
- `connexusminds-mark.svg` — network-topology mark (per `SEED/CONNEXUSMINDS-brand-design-guide.md`)
- `connexusminds-wordmark.svg` — full lockup
- `connexusminds-mark-mono.svg` — monochrome variant for single-ink contexts

### Composition (per ADR 0011)
- `connexusminds-by-hensph-lockup.svg` — first-touch commercial composition
