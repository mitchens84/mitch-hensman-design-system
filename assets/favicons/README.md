# Favicons

Favicon assets. Source: brand mark from `assets/logos/`. Generated outputs:

- `favicon.svg` — vector source (modern browsers)
- `favicon-32.png`, `favicon-16.png`, `favicon-180.png` — raster fallbacks (apple-touch-icon)
- `manifest.webmanifest` — if PWA-relevant

Generated via standard tooling (e.g. `realfavicongenerator.net` or `pwa-asset-generator`) from the mark SVG. Per-brand, mirrored to website repo `public/`.
